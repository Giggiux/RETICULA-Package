# WebServer

## How to run
You can use **sbt** to run it:

set the following Environment Variables:

```
PORT     		# The port were the server runs
DB_SERVER		# Database address of format: jdbc:postgresql://ADDRESS:PORT
DB_USER			# Db User 
DB_PASS			# Db Password
DB_NAME			# Db name
```

then 

`sbt run`
